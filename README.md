# Addressable Build Report Analyzer

Analyze Unity Addressables build reports: find duplicate assets, track cross-group dependencies, compare builds over time.

## Quick Start

```bash
cd your-unity-project/
python3 /path/to/addressable-analyzer/run.py
```

Opens at http://localhost:8080

## Features

- **Dashboard** — Overview of bundles, sizes, duplicates, cross-dependencies
- **Duplicates** — Assets duplicated across multiple bundles, sorted by severity
- **Dependencies** — Bundle dependency graph, remote-to-remote highlighted
- **Bundle Sizes** — Chart + table of all bundles sorted by size
- **Build Diff** — Compare two builds: new/resolved duplicates, size changes, added/removed bundles

## Options

```
--path PATH         Custom buildlayout.json path
--project NAME      Override project name
--project-dir DIR   Unity project directory (default: cwd)
--port PORT         Server port (default: 8080)
```

## How It Works

1. Parses `Library/com.unity.addressables/buildlayout.json` (generated by Addressables build)
2. Resolves the `rid`-based reference graph into flat, queryable data
3. Saves analysis snapshot to `~/.addressable-analyzer/reports/<project>/`
4. Serves interactive web UI
5. Click "Reload" after a new build to save another snapshot and compare

## Requirements

- Python 3.8+
- No external dependencies
